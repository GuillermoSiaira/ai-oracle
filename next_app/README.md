# AI Oracle - Next.js Frontend (Next 15 + TypeScript + Tailwind)

Setup (PowerShell):

```powershell
cd d:\projects\AI_Oracle\next_app
python -m venv .venv  # optional
npm install
npm run dev
```

Notes:
- API backends must be running: Abu Engine at http://127.0.0.1:8000 and Lilly Engine at http://127.0.0.1:8001
- Install packages: next, react, react-dom, swr, d3, recharts, tailwindcss, postcss, autoprefixer, typescript
